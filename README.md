# PhotoPrint

Бэкенд на Spring Boot. Репозиторий: https://github.com/pagadone1/ziovpoLab (ветка `zadanie2`).

Скопировать `.env.example` → `.env`. Тесты: `mvnw test -Dspring.profiles.active=test`

---

**Тема проекта:**

Сервис онлайн-заказа фото-печати — управление заказами на печать фотографий разных форматов с доставкой.

**Основные сущности:**

**Customer** (клиент) — имя, фамилия, телефон, почта; связь с заказами клиента.

**Format** (формат фото) — размер (10×15, A4 и т.д.), название, описание, цена.

**Photo** (фото) — данные о файле; связь с форматом и заказом.

**Order** (заказ) — статус, общая стоимость, дата создания; связь с клиентом, фото и доставкой.

**Delivery** (доставка) — адрес, трек-номер, статус доставки.

**Операции сервиса:**

CRUD по всем сущностям (заказы, клиенты, форматы, фото, доставка).

Бизнес-операции: создание заказа с расчётом суммы по выбранным форматам; смена статуса заказа и доставки; история заказов клиента; статистика по популярным форматам; сводка по заказам и выручке.

## Задание 2 (лицензирование)

В этой ветке также реализован модуль лицензий (таблицы `license`, `device`, `ticket` и т.д.).

API: `POST /api/license`, `/activate`, `/check`, `/renew` — см. `LicenseController`.

`Ticket` / `TicketResponse` — поля срока лицензии и ЭЦП (SHA256withRSA).
